URL Shortener

This extremely simple service redirects short urls to longer ones. The urls are provided in a csv file in the data directory.

Examples:

url            │ page                                  │ status 
───────────────+───────────────────────────────────────+─────────────────
/              │ home (this page)                      │ 200 
/dbd           │ www.gov.uk/government/publications... │ 301
/bilbo         │ not found                             │ 404
/bilbo/baggins │ not found                             │ 404
